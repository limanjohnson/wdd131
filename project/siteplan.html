Site Plan

Site Name:

Weather Companion

This name represents a web application that people can use to check the weather anywhere in the world.

Site Purpose:

The purpose of the site will allow users to search and add different locations to their "dashboard". There will be a search page, a weather dashboard for places that the user has already added, as well as a third page with interesting places that have extreme weather. Kind of like a blog page.

Scenarios:

What is the weather in my current location?

Can I save this to view it at a later date quickly?

How easy is this tool to use? Can anyone open it up and know what to do?

How should the mobile version and the desktop version look?
